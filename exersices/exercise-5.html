Try the following exercises to perfect your knowledge

  1. Create a function 'convertToFahrenheit(celsius)'
    that takes a number in degrees Celsius and returns a number in degrees Fahrenheit.
    Formula: Fahrenheit = (Celsius * 9/5) + 32
    convertToFahrenheit(25) => 77
  2. Create a function 'convertToCelsius(fahrenheit)'
    that takes a number in degrees Fahrenheit and returns a number in degrees Celsius.
    Formula: Celsius = (Fahrenheit-32)*5/9
    convertToCelsius(86) => 30.
  3. Create a function 'convertTemperature(degrees, unit)'
    that takes a number and a unit ('C' or 'F'), and converts it into the other unit.
    convertTemperature(25, 'C') => '77F'
    convertTemperature(86, 'F') => '30C'
