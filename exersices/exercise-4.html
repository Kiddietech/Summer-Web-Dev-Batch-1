Try the following exercises to perfect your knowledge

  1. Create a function called greet(); that displays the message 'Hello!' in the console. Call this function a few times.
  2. Continuing from (a) above; add a parameter called 'name' to function greet() and display the message 'Hello ${name}!'.
    Call the function a few times with different names; for example greet('Jane').
  3. Try calling the function greet() without a parameter (it will display 'Hello undefined!').
    Modify the function so that if 'name' is undefined, it should display 'Hi there' instead. Hint: if (!name)
